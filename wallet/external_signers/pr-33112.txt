
Bitcoin Core uses uses extrenal_signer -signer to interact with the external hardware wallets.
Bitcoin Core will Delegate Keys custody and transaction signing to the hardware wallet.
But will still perform PSBT construction and fee estimation. 
So, in the case the external signer is signing transactions -> but some roles are still with Bitcoin Core.
HWI -> This is a Python library and CLI that provides an interface across the hardware wallet and Bitcoin Core. 

HWI -> enumerate -> List connected devices with their fingerprints 
HWI -> getdescriptors -> Returns descriptors strings for a device 
HWI -> signtx -> signs a PSBT 
HWI -> displayaddress -> Shows an address on the hardware device 

-singer = path to hwi

Bitcoin core ----------> calls hwi enumerate --------> HWI CLI ---------> Hardware wallet
